# Python Week 2 – Data Structures & Loops

This repository contains my assignments and mini project for **Week 2** of Python practice.  
It covers lists, dictionaries, loops, file handling, and JSON/CSV operations.

---

## 📂 Contents

### Assignments
1. **List CRUD Operations**  
   - Create, Read, Update, Delete elements in a list using user input.

2. **Word Frequency Counter**  
   - Counts occurrences of each word in a sentence using a dictionary.

3. **Pattern Printing**  
   - Uses nested loops to print star (`*`) patterns.

4. **Largest & Smallest Finder**  
   - Finds the largest and smallest numbers in a list.

5. **Sorting with Lambda**  
   - Sorts student records by score using `sort()` and `lambda`.

6. **Word Counter (File I/O)**  
   - Reads a text file and counts lines, words, and characters.

7. **JSON File Reader**  
   - Reads and prints JSON data in a formatted way.
